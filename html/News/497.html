我们学习做网站，肯定少不了数据库操作。有很多朋友给我邮件，叫我帮他看看代码，怎么连部起数据库啊，怎么老是报错之类的，今天我们就以一个简易的新闻系统来学习.net读取数据库数据，接下来几天学习添加，删除和修改。其实会做新闻发布系统了就基本上掌握了数据库的增删改查了，那么很多地方就可以举一反三，做出更多的模块，再把这些模块组合起来，就成为一个简易的网站了。当然，菜鸟做的网站不可能一步到位，随着学习的深入，我们再优化代码，最后就越写越好了。
        开始之前系统必须要安装vs2008和sql2000。
        首先我们新建一个数据库test1,在里面建一张表news,字段分别为ID (int,50)其中ID表示字段名,int表示类型,50表示长度，同理Title (nvarchar，50)，Source (nvarchar,50),AddTime(datetime,8),  _Content(ntext,16),_counts(int,4)
将ID设置为主键，将标示设置为是，即自动增加。
        打开vs2008,打开菜单上的文件--新建--新建网站，在里面选择asp.net网站，在位置处选择文件系统，点击浏览，选择一个位置，这个位置就是用来存放你的网站的根目录。在语言处选择visual C# 点击确定 。

using System;
using System.Collections.Generic;
using System.Linq;
using System.Web;
using System.Web.UI;
using System.Web.UI.WebControls;
using System.Data;
using System.Data.SqlClient;
public partial class _Default : System.Web.UI.Page
{
    protected void Page_Load(object sender, EventArgs e)
    {
        if (!IsPostBack)
        {
            PageInit();
        }
    }
    /// <summary>
    ///
    /// </summary>
    protected void PageInit()
    {
        SqlConnection conn = new SqlConnection("Data Source=YYJCW-606D823C0\\SQL2000;Initial Catalog=test1;Persist Security Info=True;User ID=sa;Password=sa");
        string strSql="select * from News";//构建查询语句
        conn.Open();//打开数据库
        SqlDataAdapter sda = new SqlDataAdapter(strSql, conn);//实例化适配器
        DataSet ds=new DataSet();//实例化DataSet 
        sda.Fill(ds,"news");//将查询结果装进ds里
        conn.Close();//关闭数据库
        Repeater1.DataSource = ds;//给Repeater1指定数据源
        Repeater1.DataBind();//将数据源绑定到Repeater1上
    }
}
